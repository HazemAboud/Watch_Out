# 🚀 Watch_Out: YOLOv11 Obstacle Detection System (Group Project)

**Watch_Out** is a computer vision project designed for detection and hazard assessment using **YOLOv11** architecture, this project provides tools for hyperparameter optimization, interactive training dashboards, model benchmarking, and a realtime detection GUI.

## 🌟 Key Features

*   **Fine Tuned Model**: Fine tuned *YOLO11s* model on Person, Animal, Car, Computer Vision Dataset with 7500 samples and acheived 0.558 mAP@50 on the complex dataset.

*   **Real-Time Hazard Assessment**: Distance estimation based on frame occupancy percentage, providing visual alert for "SAFE", "CAUTION", and "DANGER" levels.

*   **Training GUI**: Interactive dashboard for monitoring training progress, visualizing metrics, and adjusting hyperparameters using streamlit.
